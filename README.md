# SMSWatchRinger（安卓短信关键词告警）

用于：监控手机短信，命中关键词/正则后，弹窗 + 通知 +（可选）铃声/语音播报第一时间提醒。

## 功能

- 短信监控：监听系统短信广播（`SMS_RECEIVED`），读取短信内容
- 规则：关键词、指定号码、排除词、正则
- 防重复：同内容冷却（X 秒内只提醒一次）
- 推送：可选将命中的短信 POST 到服务端接口（JSON）
- 告警：命中弹窗（前台）、高优先级通知、连续响铃（可开关）、语音播报（TTS）
- 关键词级别配置（同一条短信取“第一个命中关键词”的配置）
  - 播报文本：`关键词=>播报内容`
  - 是否响铃：`| 响铃=开/关`
  - 重播次数：`| 重播=1-5`
  - 示例：`交警=>交警提醒 | 响铃=关 | 重播=3`

## 使用

1. 用 Android Studio 打开本项目根目录
2. 安装到手机后，打开 App：
   - 授权「短信权限」
   - Android 13+ 额外授权「通知权限」
3. 进入「设置」配置关键词与告警方式

## 构建（Debug APK）

```bash
./gradlew assembleDebug
```

## 说明

- 仅本地处理短信内容，不上传服务器
- 部分手机系统会限制后台行为；如发现不触发，优先检查：
  - 权限是否已授权
  - 是否被省电策略限制（加入白名单）
